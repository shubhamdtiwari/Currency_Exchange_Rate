# 💱 Currency Exchange Rate

A modern and responsive currency converter built using **React**, **Vite**, and **Tailwind CSS**. The application allows users to convert currencies using real-time exchange rates through a clean and intuitive interface.

🌐 **Live Demo:** https://changemycurrency.netlify.app/

---

## ✨ Features

- 🌍 Convert between multiple international currencies
- 📈 Real-time exchange rates
- 🔄 One-click currency swap
- ⚡ Fast performance with Vite
- 📱 Fully responsive design
- 🎨 Clean and modern UI
- ❌ Input validation for better user experience

---

## 🛠️ Tech Stack

- React.js
- Vite
- Tailwind CSS
- JavaScript (ES6+)
- HTML5
- CSS3

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/shubhamdtiwari/Currency_Exchange_Rate.git
```

### Navigate to the project

```bash
cd Currency_Exchange_Rate
```

### Install dependencies

```bash
npm install
```

### Run the development server

```bash
npm run dev
```

### Build for production

```bash
npm run build
```
